### Lógicos ou booleanos

Os valores lógicos, chamados de Booleanos, representam verdadeiro e falso.

```ruby
evento_lindo = true
```

‘**True**’ é verdadeiro em inglês, idioma que a linguagem ruby utiliza.

```ruby
evento_feio = false
```

‘**False**’ é falso em inglês.

```ruby
(irb)> true != false
# true
(irb)> !true
# false
(irb)> tutora_feliz == 1
# false
(irb)> false.to_s
# "false"
```
